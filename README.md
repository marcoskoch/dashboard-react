## Instalação

Faça o clone do nosso repositório e utilize os comandos:

```sh
$ git clone https://github.com/marcoskoch/dashboard-react.git
$ cd dashboard-react
$ npm install
$ npm start
```

Acesse a página no link: http://localhost:3000
